# Todo-list

Работа с ООП, ES6 Модулями. Пятый месяц обучения.  <br />
[Ссылка на задание](https://www.theodinproject.com/lessons/node-path-javascript-todo-list)  <br />
[Ссылка на Pages](https://artemmakhaydinov.github.io/todo-list/)

Планировщик задач. С помощью библиотеки date-fns с течением времени динамически сдвигает дни и недели, а так же обновляет срок выполнения задач. Данные хранятся в localStorage. В проекте использовался webpack.
